# SQL-Project-Job-Posting-Analysis

This project consists of a series of queries that builds up to finding the optimal skills to learn. 
In this case it's for data analyst roles. These queries navigate through a database with over 700,000 
job postings. 

Query 1 is about finding the top paying jobs. 
Query 2 is about finding the skill that the top paying job demands most. 
Query 3 is about finding the top demanded skills.
Query 4 is about finding the top paying skills. 
Query 5 is about finding the top demanded and top paying skills (most optimal).
